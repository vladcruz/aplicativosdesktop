# 💻 **Desenvolvimento de Sistemas - C# e .NET**

Bem-vindo ao repositório oficial da disciplina de Desenvolvimento de Sistemas.

Este repositório contém todos os códigos-fonte, exemplos práticos e gabaritos dos desafios desenvolvidos em sala de aula.

O objetivo do nosso curso é levar você além do código básico: vamos focar em Código Limpo (Clean Code), dominar a Arquitetura Orientada a Objetos (POO), processar dados com LINQ e persistir informações no disco (Arquivos e Bancos de Dados).

## 🛠️ Tecnologias Utilizadas

**Linguagem:** C# (C-Sharp)

**Plataforma:** .NET 6.0 (Cross-Platform)

**IDE:** Visual Studio 2022 Community

**Paradigma:** Orientação a Objetos (POO)

## 📚 Índice de Aulas e Códigos

Abaixo você encontra a separação de todos os conteúdos abordados no semestre. Clique nos links para acessar os arquivos de código correspondentes.

### Aula 01: Fundamentos Modernos e Modularização

Nesta aula, revisamos a base do C# e entendemos o conceito do Princípio DRY (Não se repita). Transformamos códigos procedimentais longos em sistemas limpos divididos por funções (Métodos Estáticos).

📄 Calculadora Modular (Exemplo de Aula)

🎯 Desafio: Sistema de Folha de Pagamento

### Aula 02: POO I - A Anatomia da Classe

Saímos do paradigma estruturado e entramos no mundo dos Objetos. Aprendemos a criar nossos próprios tipos de dados, protegendo informações vitais através do Encapsulamento e dos Construtores.

📄 Conta Bancária Segura (Exemplo de Aula)

🎯 Desafio: Modelagem de Item de RPG

### Aula 03: POO II - Herança, Polimorfismo e Robustez

Como evitar a repetição de código entre classes parecidas? Aprendemos a usar Herança, sobrescrever métodos com Polimorfismo, padronizar com Interfaces e proteger o sistema contra quebras usando try/catch/throw.

📄 Sistema Bancário Avançado (Exemplo de Aula)

🎯 Desafio: Gestão de Frota (Carros e Caminhões)

### Aula 04: Coleções e Manipulação de Dados (LINQ)

Dissemos adeus aos Arrays de tamanho fixo. Mergulhamos nas Listas (List<T>), Dicionários (Dictionary<K,V>), Filas e Pilhas. Também aprendemos a mágica do LINQ para consultar dados de forma declarativa e em uma única linha.

📄 Exemplos de Listas, Filas e Dicionários

📄 O Poder do LINQ (Exemplo de Aula)

🎯 Desafio: Spotify Data Analytics

### 👾 Aula 05: Laboratório Prático (Space Invaders)

Uma aula especial e 100% prática! Aplicamos todos os conceitos de POO (Herança, Entidades) e LINQ (Detecção de colisão) na construção de um motor de jogo em tempo real no próprio Console do C#.

🎮 Boilerplate (Código base para alunos)

🏆 Gabarito: Space Invaders Completo

### Aula 06: Persistência de Dados e a Linguagem das IAs

Aprenda a salvar os dados dos seus objetos no Disco Rígido para que eles não sumam ao fechar o programa. Exploramos o ciclo de leitura e gravação em formatos essenciais do mercado e para treinamento de Inteligência Artificial.
